Intro

Lists and Dictionaries can be useful in a complicated program. Rather than storing multiple variables to hold similar data, you can create one list and store the data in there.

Python uses Lists and Dictionaries, while C++ calls them Vectors and Maps.
In a programming class for college, you are likely to be writing your own versions of Lists, Dictionaries, and more. These more complex data types are called Data Structures.

Lists vs. Vectors

We don't need to import any libraries to use Lists in Python, but to use a Vector in C++, we need to include its library file: 

#include <vector>

In Python, the data-types are not concrete, so you can create a list and store any type of object in it immeditely.
In C++, your Vector must hold all of one type of item. For example, ints, floats, strings, or bools.
They could even store class objects that you've created.

Lists vs. Vectors: Creating a List/Vector

In Python:

itemPrices = []
studentNames = []

In C++:

vector<float> itemPrices;
vector<string> studentNames;

C++ uses templates to allow you to create a class that can handle different data-types, rather than hardcoding a FloatListObject, StringListObject, etc.

Templates are taught later in Programming II, usually.

Lists vs. Vectors: Insertion

In Python, we can insert data into a list like this:

studentNames.append( "Kourtney" )
studentNames.append( "Frank" )

itemPrices.append( 5.95 )
itemPrices.append( 19.99 )

In C++, we can do the same with:

studentNames.push_back( "Kourtney" );
studentNames.push_back( "Frank" );

itemPrices.push_back( 5.95 );
itemPrices.push_back( 19.99 );

To get the size of your list in Python, you can use: 

len( studentNames )

To get the size of your vector in C++, you can use: 

studentNames.size();

Lists vs. Vectors: Accessing Values

When you store data in a List or Vector, you can access that data directly by using the subscript operator, [ ].

Python:

studentNames.append( "Kourtney" )
studentNames.append( "Frank" )
studentNames.append( "Patel" )
studentNames.append( "Hana" )

print( "Student " + studentNames[3] + " Report Card" )

C++:

studentNames.push_back( "Kourtney" );
studentNames.push_back( "Frank" );
studentNames.push_back( "Patel" );
studentNames.push_back( "Hana" );

cout << "Student " << studentNames[3] << " Report Card" << endl;

Lists vs. Vectors: Deletion

If you want to delete an item at the index i in Python: 

studentNames.pop( i )

If you want to delete an item at the index i in C++: 

studentNames.erase( studentNames.begin() + i );
Dictionaries vs. Maps

To use a Map in C++, we must include the library: 

#include <map>

Dictionaries vs. Maps: Creating a Dictionary/Map

In Python:

gradeLookup = {}

In C++:

map<string, int> gradeLookup;

As with anywhere else in C++, we must define the data types we're using. For our map, the key will be a string (the student's name), and the value will be their grade (0 - 100).

Dictionaries vs. Maps: Insertion

In Python, we can insert values into the Dictionary by assigning a value to the key... 

gradeLookup["Elizabeth"] = 100
gradeLookup["Anne"] = 75
gradeLookup["Aki"] = 86

In C++, we can do the same thing with insert, but we also have to specify that we're inserting a pair of values: 

gradeLookup.insert( pair<string, int>( "Elizabeth", 100 ) );
gradeLookup.insert( pair<string, int>( "Anne", 75 ) );
gradeLookup.insert( pair<string, int>( "Aki", 86 ) );

This is another place where templates are being used, since we wouldn't want to have to declare a StringIntPair, StringStringPair, IntIntPair, and so on for each type of Key-Value pairs we could possibly have.

We can find the size of our Dictionary/Maps the same way as with Lists/Vectors:

Python:

len( gradeLookup )

C++:

gradeLookup.size();

Dictionaries vs. Maps: Accessing Values

We can access the values in our Dictionary/Map with the subscript operator [ ].

Python:

currentStudent = "Elizabeth"
print( "Student: " + currentStudent )
print( "Grade: " + int( gradeLookup[ currentStudent ] ) )

C++:

string currentStudent = "Elizabeth";
cout << "Student: " << currentStudent << endl;
cout << "Grade: " << gradeLookup[ currentStudent ] << endl;
Arrays in C++

Beyond the more complex data structures the STL (Standard Template Library) provides for you, the most basic form of lists in C++ is the static array.

These arrays are declared similar to most variables in C++, except you use the subscript operator: 

int myNumbers[3];
string studentNames[20];

Simple arrays like this cannot be resized. If you wish to resize an array like this, you would you have to create a second array and copy the values in. 

string studentNames[20];
string newStudentNames[30];

for ( int i = 0; i < 20; i++ )
{
	newStudentNames[i] = studentNames[i];
}

You can access values from the array directly with the subscript operator, like: 

string studentNames[20];

for ( int i = 0; i < 20; i++ )
{
	cout << "Good morning, " << studentNames[i] << endl;
}

Think of the subscript operator literally like subscripts in algebra, when you have an array.

x0 , x1, x2, ..., xn
