What are classes?

Classes are a means for us to create our own data-type, often storing multiple data-types inside of itself, as well as functionality. For example:

  • A television object might have variables for Current Channel, Volume Level, and On/off Toggle.
    Its functionality includes Turn On, Turn Off, Channel Up, Channel Down, Go To Channel (number), Volume Up, Volume Down.
  • A book object might have variables such as Title, Author, and Current Page.
    The book would contain page objects that store lines of text on it.
    You'd use the book's Turn Page Forward and Turn Page Back functionality to go through the "list" of pages available.
  • An enemy object in a video game may have variables like Speed, Direction, HP (Hit points), and Strength (for when it attacks).
    Functions of the enemy might be Walk, Attack, Look for Player, and Change Direction.
    If you've played Super Mario Bros., think about how some enemies keep moving, even if they fall down a pit. Some enemies will turn around and go the other direction if they run out of floor. Behavior would be part of the enemy object.
