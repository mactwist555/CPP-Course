Types of Conditional Statements

Whether in Python or C++, you may want to check whether one of these statements is true or false:

Variable1 is equal to Variable2
Variable1 is not equal to Variable2
Variable1 is less than Variable2
Variable1 is less than or equal to Variable2
Variable1 is greater than Variable2
Variable1 is greater than or equal to Variable2
Variable1 is true
Variable1 is false

We can check these in the following ways:

Multiple Conditional Statements

You may also check for whether multiple conditions exist simultaneously (and), or if one of two conditions are true (or).

Python:

if ( age > 18 and age < 21 ):
	print( "You can vote, but you cannot drink!" )

if ( grade == "F" or grade == "D" ):
	print( "You failed :(" )

C++:

if ( age %gt; 18 && age < 21 )
{
	cout << "You can vote, but you cannot drink!" << endl;
}

if ( grade == "F" || grade == "D" )
{
	cout << "You failed! :(" << endl;
}

In C++, the and operator is &&, while the or operator is || (usually shift + \)

If Statements

In Python, if statements look like:

age = input( "What is your age? " )

if ( age < 16 ):
	print( "You're a kid!" )
elif ( age == 16 ):
	print( "Did you get your drivers license yet?" )
elif ( age > 16 and age < 21 ):
	print( "You can drive, but you can't drink yet!" )
else:
	print( "Yay, you're old!" )

In Python, anything that belongs within an if condition is indented by one tab. In C++, the amount of tabs does not matter, it contains the logic for that condition between curly braces { }.

Curly braces are common in C++ to enclose data - logic, declarations, etc.
main() is a function and has curly braces to denote the beginning and end of it. 

int age;
cin >> age;

if ( age < 16 )
{
	cout << "You're a kid!" << endl;
}

else if ( age == 16 )
{
	cout << "Did you get your drivers license yet?" << endl;
}

else if ( age > 16 && age < 21 )
{
	cout << "You can drive, but you can't drink yet!" << endl;
}

else
{
	cout << "Yay, you're old!" << endl;
}

Notice that the if statement doesn't end with a semi-colon. An if statement is more than one line of code, same with a function body and a class delcaration.

Also notice that rather than the keyword and, C++ uses double ampersands (&&) for AND, and it would use double pipes (||) for OR.

Switches

C++ also has switch statements, which are similar to if statements but only work for integer values. 

cout << "Choose a menu option:" << endl;
cout << "1. Play game" << endl;
cout << "2. View Help" << endl;
cout << "3. Quit" << endl;

int choice;
cin >> choice;

switch( choice )
{
	case( 1 ):
		PlayGame();	// Function call
	break;
	
	case( 2 ):
		DisplayHelp();	// Function call
	break;
	
	case( 3 ):
		return 0;	// Quit program
	break;
	
	default:
		cout << "Invalid option!" << endl;
	break;
}

We first need to create a switch statement, specifying which variable we're investigating (here, it's "choice").

Then, we set up several possible options, 1, 2, or 3.
"default:" is the case if nothing matches.

If you don't include break;, then the program will execute the next case.

Nesting if statements

In both languages you can nest if statements, or if statements loops, etc.

Python:

if ( playerAlive == True ):
	
	if ( state == "battle" ):
		print( "A: Attack" )
		print( "B: Block" )
		print( "D: Dodge" )
		print( "R: Run away" )
	
	elif ( state == "navigating" ):
		print( "N: North" )
		print( "S: South" )
		print( "E: East" )
		print( "W: West" )
		
	choice = input( "What will you do next?" )
		
else:
	print( "Game Over" )

C++:

if ( playerAlive == true )
{
	
	if ( state == "battle" )
	{
		cout << "A: Attack" << endl;
		cout << "B: Block" << endl;
		cout << "D: Dodge" << endl;
		cout << "R: Run away" << endl;
	}
	else if ( state == "navigating" )
	{
		cout << "N: North" << endl;
		cout << "S: South" << endl;
		cout << "E: East" << endl;
		cout << "W: West" << endl;
	}
	
	cout << "What will you do next? ";
	cin >> choice;
}
else
{
	cout << "Game Over" << endl;
}
