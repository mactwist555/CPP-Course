While Statements

Let's say that we were writing a program that would continuously loop until the user decided to quit, which looked something like: 

	Please choose a command:
	1. Add new event to schedule
	2. Review agenda
	3. Save and Exit

Here, you would use a boolean value to keep track of when the user wanted to quit ("doneWithProgram"), and keep looping the program until "doneWithProgram" was true.

Python:

doneWithProgram = False

while ( not doneWithProgram ):
	print( "Please choose a command:" )
	print( "1. Add new event to schedule" )
	print( "2. Review agenda" )
	print( "3. Save and Exit" )

In C++, we would have to declare a bool type, and our "true/false" values would be all lower-case. 

bool doneWithProgram = false;

while ( !doneWithProgram )
{
	cout << "Please choose a command:" << endl;
	cout << "1. Add new event to schedule" << endl;
	cout << "2. Review agenda" << endl;
	cout << "3. Save and Exit" << endl;
}

While statements look pretty similar either way. You can put any statement as the loop's conditional, so long as it can be evaluated to True or False in the end...

Python:

age = input( "Enter your age" )

# Don't let the program continue until the user provides a valid age
while ( age < 0 or age > 120 ):
	print( "Invalid age, your age must be a positive number between 0 and 120 years.
	age = input( "Please re-enter your age: " )

C++:

int age;
cin >> age;

// Don't let the program continue until the user provides a valid age
while ( age < 0 || age > 120 )
{
	cout << "Invalid age, your age must be a positive number between 0 and 120 years." << endl;
	cin >> age;
}
For Loops

For-Each loops are kind of icky in C++98, so we are going to skip those. You can achieve the same thing with a normal for loop, and iterating over a list from 0 to the size of the list.

Python:

listStudents = []
amountOfStudents = input( "How many students? " )

# Get student names
for ( i in range( 0, int(amountOfStudents) ):
	name = input( "What is the name of student " + str( i ) + "? " ) 
	listStudents.append( name )
	
# Output student names
for ( student in listStudents ):
	print( student )

C++:

vector<string> listStudents;
int amountOfStudents;
cin >> amountOfStudents;

// Get student names
for ( int i = 0; i < amountOfStudents; i++ )
{
	string name;
	cout << "What is the name of student " << i << "? ";
	cin >> name;
	listStudents.push_back( name );
}

// Output student names
for ( int i = 0; i < listStudents.size(); i++ )
{
	cout << listStudents[i] << endl;
}

With C++, our for loops look quite different. Python is formed like:

for variable_index in range( start_point, end_point ):

But with C++, it's formed this way: 

for ( int variable_index = start_point; variable_index < end_point; variable_index++ ):

The last argument is how much the variable_index hops forward each loop. This could be addition (variable_index++ to add one, variable_index += 5 to add five), or any other math command (variable_index *= 2 for multiply by 2 each time).

Then, when we're accessing elements in the listStudents list, we have to call the list by name (listStudents) and then use the subscript operator [ ] to access the element at that specific index.
