Output

In Python, we can display a message with 

print( "Hello!" )

In C++, we utilize the iostream library and use:

cout << "Hello!" << endl;

C++ handles I/O as a stream, indicated by << or >>. Think of the "Hello!" message and the endl (end-line) command as being streamed TO the cout (console-out).

You can display just an endl:

cout << endl << endl;

Or you can write text without the end-line, and any text that comes afterwards will be on the same line: 

cout << "Enter your name: ";
Variables

In Python, to make a variable you just start using it...

name = "Bob"
age = input( "Enter your age: " )

But in C++, we must declare our variables first. Additionally, they can only be one data-type, such as a string, int, float, or bool.

It's possible to convert between, but you would have to store it in a new variable with the right data-type. 

string name;
int age;
float cost;
Input

In Python, we can get input and specify a message using...

name = input( "What is your name? " )

In C++, we will display the message separately, then use the cin (console-in) command: 

	string name;
	cout << "What is your name? ";
	cin >> name;

Notice how cin has >> and cout has <<.
Think of this as the way the streams flow; the message flows OUT to the console-out, and the value of 'name' flows IN, from the cin command to the name variable.

In Python, if the user entered a value and you wanted to use it as an integer, you would have to cast it: 

age = input( "Please enter your age: " )
oldAge = int(age) + 1

And if you wanted to print an int out you would use 

print( oldAge )

or to add it to a string message, you have to cast it back to a string: 

print( "In a year you will be " + str(oldAge) )

In C++, when you get input and store it in a variable you won't have to cast it: 

// Declare variables
string name;
int age;
float cost;

// Get input from user
cout << "Name? ";
cin >> name;

cout << "Age? ";
cin >> age;

cout << "Cost? ";
cin >> cost;

// Display results
name = "Captain " + name;		// Concatenating strings
age = age + 1;				// Adding integers
cost = cost + ( cost * 0.725 );		// Adding floats
NOTE: You can add a float to an int, but the decimal value will be lost.
Outputting Variables

In C++, you can output all the values like this:

cout << "You are " << name << " and you are " << age << " years old and "
	<< "sodas cost $" << cost << endl;

Notice how the line is broken. You can stop a line in C++ (don't put the ;) and continue it on the next line. The tab isn't even required, but I put it there to help specify that it's part of the cout above.
